Weather API

How to get the data for a particular station on a particular day:

URL format: http://127.0.0.1: 5000/api/v1/stationid/date

URL Example: http://127.0.0.1: 5000/api/v1/22/1988-10-31
Yes, the dashes (-) ARE mandatory!
